# 1. Levantamento de Requisitos e Regras de Negócio
# Requisitos Funcionais (RF)
RF01 - Manter Filmes: O sistema deve permitir o cadastro de filmes com título, gênero, diretor, elenco e duração.

RF02 - Manter Sessões: O sistema deve permitir criar sessões vinculando um filme a uma sala em um horário específico.

RF03 - Registrar Público: O sistema deve permitir o registro da quantidade de espectadores presentes em uma sessão.

RF04 - Totalização de Público: O sistema deve gerar relatórios de público total por filme e por unidade de cinema.

Regras de Negócio (RN)
RN01 - Choque de Horários: Uma sala não pode ter duas sessões simultâneas.

RN02 - Intervalo de Limpeza: Deve haver um intervalo mínimo de 20 minutos entre o fim de uma sessão e o início da próxima.

RN03 - Capacidade Máxima: O registro de público não pode ultrapassar a lotação máxima da sala.
